# Screenshots Folder

Place your ApexLog app screenshots here.

## Naming Convention

Name your screenshots:
- `screenshot-1.jpg` (or `.png`)
- `screenshot-2.jpg`
- `screenshot-3.jpg`
- `screenshot-4.jpg`
- `screenshot-5.jpg`

You can add more screenshots by adding more carousel items in `index.html`.

## Image Requirements

- **Format**: JPG or PNG
- **Aspect Ratio**: 9:16 (vertical phone screenshots)
- **Recommended Size**: 390px wide Ã— 844px tall (or maintain 9:16 ratio)
- **File Size**: Optimize for web (under 500KB each recommended)

## Tips

- Use consistent aspect ratio for all screenshots
- Optimize images before adding them (use tools like TinyPNG or ImageOptim)
- Screenshots will appear in the order they're listed in `index.html`
